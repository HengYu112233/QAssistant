# 视频演示快速检查清单

使用此文档在录制视频前确保项目状态正确。

## 录制前必做（5 分钟）

## ✅ 环境检查

- [ ] Python 虚拟环境已激活：`.venv\Scripts\Activate.ps1`
- [ ] 依赖已安装：`pip list | grep -E "aiohttp|aiosqlite|openai|websockets"`
- [ ] `.env` 文件已配置：`LLM_API_KEY` 和 `LLM_MODEL` 已填入
- [ ] NapCat 已启动且监听 `ws://127.0.0.1:3001`
- [ ] 可网络访问火山引擎 Ark：http://ping.volces.com 返回 200

## ✅ 项目状态重置

```powershell
# 清空数据库和记忆
Remove-Item -Path ".\data\*.db*" -Force -ErrorAction SilentlyContinue
Remove-Item -Path ".\data\ai_memory" -Recurse -Force -ErrorAction SilentlyContinue

# 验证状态
Get-ChildItem .\data\
```

预期结果：`data` 目录为空或不存在，这表示项目处于初始化状态。

- [ ] 所有 `.db*` 文件已删除
- [ ] `ai_memory` 目录已删除
- [ ] 项目处于"零数据"初始化状态

## ✅ 代码检查

- [ ] 没有 `print()` 语句遗留（仅使用 `logger`）
- [ ] 没有硬编码的 API Key（均通过 `.env` 加载）
- [ ] 没有 TODO 注释遗留（或已沟通清楚）

## 常见问题应急方案

| 问题 | 应急方案 |
|------|--------|
| 机器人未回复 | 检查 `.env` 配置，重启 NapCat，重新运行 `python main.py` |
| API 超时 | 等待 3-5 秒重试，或切换网络（如关闭 VPN 再开） |
| 数据库锁定错误 | `Ctrl+C` 关闭程序，删除 `data/*.db-*` 文件，重新运行 |
| 消息未被识别 | 确认输入格式（如记账需包含金额），参考 `DEMO_SCRIPT.md` |
